# موقع خِدْمتي - Khedmati

منصة إلكترونية وسيطة تساعد المستخدمين على طلب وإنجاز الخدمات الحكومية والخاصة بسهولة، بدون الحاجة لفهم الإجراءات المعقدة.

## ✨ المميزات

### واجهة المستخدم
- **تصميم عصري**: واجهة حديثة تعكس ثقة وكفاءة المنصة
- **تجربة مستخدم سلسة**: تصميم متجاوب مع جميع الأجهزة
- **تفاعلية عالية**: مؤثرات بصرية وحركات سلسة
- **دعم كامل للغة العربية**: واجهة عربية احترافية

### الوظائف الأساسية
- **نظام طلب الخدمة**: واجهة سهلة لاختيار الخدمة ورفع المستندات
- **تتبع الطلبات**: متابعة لحظية لحالة الطلبات
- **لوحة تحكم إدارية**: إدارة كاملة للطلبات والمستخدمين
- **نظام بحث متقدم**: البحث السريع عن الخدمات

## 🛠️ التقنيات المستخدمة

### Frontend
- **HTML5**: بنية واضحة ومعيارية
- **Tailwind CSS**: تصميم حديث وسريع
- **JavaScript**: تفاعلية ووظائف ديناميكية
- **Anime.js**: مؤثرات بصرية وحركات
- **Font Awesome**: أيقونات عالية الجودة

### الخطوط والتصميم
- **Cairo**: خط عربي حديث للعناوين
- **Inter**: خط عالمي للنصوص
- **Tajawal**: خط للأرقام والبيانات

## 🚀 كيفية التشغيل

### الطريقة الأولى: باستخدام Python
```bash
# الانتقال إلى مجلد المشروع
cd /mnt/okcomputer/output

# تشغيل خادم ويب محلي
python -m http.server 8000

# فتح المتصفح وزيارة
http://localhost:8000
```

### الطريقة الثانية: باستخدام Node.js
```bash
# تثبيت خادم ويب بسيط
npm install -g http-server

# تشغيل الخادم
http-server

# فتح المتصفح وزيارة
http://localhost:8080
```

### الطريقة الثالثة: فتح الملفات مباشرة
يمكن فتح ملف `index.html` مباشرة في المتصفح.

## 📱 الصفحات الرئيسية

### 1. الصفحة الرئيسية (index.html)
- **الهيدر**: شعار الموقع وقائمة التنقل
- **قسم البطل**: عنوان رئيسي ووصف الخدمة مع زر الدعوة للعمل
- **الخدمات**: عرض البطاقات التفاعلية للخدمات
- **كيف تعمل**: خطوات العمل مع رسوم توضيحية
- **الإحصائيات**: أرقام توضح نجاح المنصة
- **آراء العملاء**: تقييمات ومراجعات العملاء

### 2. صفحة الخدمات (services.html)
- **شريط البحث**: بحث لحظي عن الخدمات
- **فلاتر التصنيف**: تصفية حسب النوع (حكومية، بنكية، تعليمية، مرورية)
- **بطاقات الخدمات**: عرض تفصيلي لكل خدمة مع السعر ومدة الإنجاز
- **نموذج الطلب**: نافذة منبثقة لطلب الخدمة

### 3. صفحة تتبع الطلبات (track.html)
- **نموذج البحث**: البحث برقم الطلب أو رقم الهاتف
- **معلومات الطلب**: تفاصيل الطلب والعميل
- **شريط التقدم**: عرض مرئي لمراحل الإنجاز
- **سجل التحديثات**: تسلسل زمني لكل تغييرات حالة الطلب

### 4. لوحة التحكم الإدارية (admin.html)
- **الإحصائيات**: نظرة عامة على أداء المنصة
- **جدول الطلبات**: إدارة الطلبات مع إمكانية التعديل والحذف
- **الإجراءات السريعة**: أدوات مساعدة للإدارة

## 🔧 التخصيص

### تغيير الألوان
يمكن تغيير الألوان من ملف CSS الموجود في كل صفحة:
```css
:root {
    --primary-blue: #1e3a8a;    /* اللون الرئيسي */
    --success-green: #10b981;   /* لون الإنجاز */
    --light-gray: #f8fafc;      /* الخلفية الفاتحة */
    --dark-gray: #374151;       /* النصوص الداكنة */
}
```

### إضافة خدمات جديدة
في صفحة `services.html`، يمكن إضافة بطاقات خدمات جديدة:
```html
<div class="service-card bg-white rounded-2xl p-6 shadow-lg border border-gray-100" data-category="government" data-name="اسم الخدمة">
    <div class="w-16 h-16 bg-blue-100 rounded-xl flex items-center justify-center mb-6">
        <i class="fas fa-icon text-blue-600 text-2xl"></i>
    </div>
    <h3 class="text-xl font-semibold text-gray-900 mb-3">اسم الخدمة</h3>
    <p class="text-gray-600 mb-4">وصف الخدمة</p>
    <div class="flex items-center justify-between mb-4">
        <span class="text-sm text-gray-500">مدة الإنجاز: X أيام</span>
        <span class="text-sm font-semibold text-green-600">من XX جنيه</span>
    </div>
    <button class="w-full btn-primary text-white py-3 rounded-lg font-medium" onclick="requestService('service-id')">
        اطلب الخدمة
    </button>
</div>
```
