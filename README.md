## ✨ الميزات الرئيسية

### 🎨 تصميم طفولي جذاب
- **ألوان مبهجة**: تدرجات لونية جميلة (أخضر زمردي، تركواز، سماوي)
- **رسومات متحركة**: أيقونات تفاعلية وحركات سلسة
- **واجهة بسيطة**: سهلة الاستخدام للأطفال
- **وضع داكن**: دعم كامل للوضع الداكن مع ألوان مريحة للعين
- **ثيمات متعددة**: 4 ألوان مختلفة (تركواز، أخضر، بنفسجي، أزرق)

### 🏆 نظام المكافآت والتحفيز
- **نجوم**: احصل على نجمة ⭐ عند إتمام كل سورة
- **رسائل تشجيعية**: رسائل عربية محفزة عند الإنجاز
- **شاشة احتفالية**: نافذة متحركة عند إكمال السورة
- **تتبع التقدم**: شريط تقدم يوضح نسبة الإنجاز الكلي

### 📚 ميزات تعليمية
- **5 سور قصيرة**: الفاتحة، العصر، الإخلاص، الفلق، الناس
- **تلاوة صوتية**: بصوت القارئ مشاري العفاسي
- **تظليل الكلمات**: تتبع الكلمات أثناء التلاوة
- **تفسير مبسط**: شرح سهل لمعاني الآيات
- **مساعد ذكي**: اسأل أي سؤال عن السورة باستخدام Gemini AI

### 🔍 البحث والتصفح
- **بحث ذكي**: ابحث بالاسم العربي أو الإنجليزي أو رقم السورة
- **علامات مرئية**: 
  - ✅ علامة خضراء للسور المكتملة
  - 🎯 شارة "جاري القراءة" للسورة الحالية
  - شريط تقدم لكل سورة

### 💾 حفظ التقدم
- **حفظ تلقائي**: يحفظ تقدمك في المتصفح
- **استمرارية**: ارجع في أي وقت وأكمل من حيث توقفت
- **إحصائيات**: تتبع عدد الآيات المكتملة والنجوم المكتسبة

### 📱 متوافق مع جميع الأجهزة
- **Responsive Design**: يعمل بشكل مثالي على الموبايل والتابلت والكمبيوتر
- **أداء سريع**: تحميل سريع وتجربة سلسة

## 🚀 التشغيل المحلي

### المتطلبات
- Node.js (الإصدار 16 أو أحدث)
- npm أو yarn

### خطوات التشغيل

1. **استنساخ المشروع**
```bash
git clone https://github.com/your-username/Quran-for-Kids.git
cd Quran-for-Kids
```

2. **تثبيت المكتبات**
```bash
npm install
```

3. **إعداد مفتاح Gemini API**
   - أنشئ ملف `.env.local` في المجلد الرئيسي
   - أضف مفتاح API الخاص بك:
```
VITE_GEMINI_API_KEY=your_api_key_here
```
   - احصل على مفتاح مجاني من: [Google AI Studio](https://makersuite.google.com/app/apikey)

4. **تشغيل التطبيق**
```bash
npm run dev
```

5. **افتح المتصفح**
   - انتقل إلى: `http://localhost:5173`

## 🎯 كيفية الاستخدام

### للأطفال 👶

1. **اختر سورة** من القائمة الرئيسية
2. **اضغط على زر التشغيل** ▶️ لسماع التلاوة

- **تتبع التقدم**: راقب عدد السور المكتملة والنجوم

## 🔄 نظام التقدم

### كيف يعمل؟

1. **تتبع الآيات**: يتم تسجيل كل آية تستمع إليها
2. **حساب النسبة**: `(الآيات المكتملة / إجمالي الآيات) × 100`
3. **إكمال السورة**: عند الوصول للآية الأخيرة
4. **منح النجمة**: نجمة واحدة لكل سورة مكتملة
5. **الحفظ التلقائي**: يحفظ في localStorage

### وظائف الخدمة

```typescript
// الحصول على التقدم
const progress = getProgress();

// تحديث السورة الحالية
updateCurrentSurah(surahId);

// تحديث تقدم السورة
updateSurahProgress(surahId, percentage);

// إكمال السورة ومنح نجمة
completeSurah(surahId);

// إعادة تعيين التقدم
resetProgress();
```
